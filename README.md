# YLink Protocol

YLink is a custom high-speed communication protocol designed to support half-duplex transfer using only 6 physical pins

## Physical Interface 
| Pin | Function |
|----|---------  |
| P0 | Clock	 |
| P1 | CTRL0 	 |
| P2 | CTRL1 	 |
| P3 | ACK/NACK  |
| P4 | DATA1 	 |
| P5 | DATA2 	 |
